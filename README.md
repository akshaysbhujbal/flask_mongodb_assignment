# Flask MongoDB Atlas Assignment

This project is a Flask web application integrated with MongoDB Atlas.

## Features

- Flask API route (`/api`)
- Fetch data from MongoDB Atlas
- Form submission using Flask
- Store data in MongoDB Atlas
- Success and error handling

## Technologies Used

- Python
- Flask
- MongoDB Atlas
- HTML
- CSS

## Run Project

```bash
python3 app.py
```

Open in browser:

```text
http://127.0.0.1:5000
```

API Route:

```text
http://127.0.0.1:5000/api
```
